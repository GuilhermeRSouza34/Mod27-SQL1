# Projeto de Criação de Tabela e Uso de Comandos DDL e DML

Este projeto envolveu a criação de uma tabela em um banco de dados relacional utilizando comandos da linguagem DDL (Data Definition Language) para definir a estrutura da tabela. Além disso, foram utilizados comandos DML (Data Manipulation Language) para inserir, atualizar e consultar dados na tabela criada. O objetivo foi demonstrar o uso prático e eficaz desses comandos para gerenciar esquemas e dados no banco de dados.

## Tecnologias Utilizadas

- **Banco de Dados Relacional**: Utilizado para criação da tabela e manipulação dos dados, possivelmente PostgreSQL, MySQL, ou similar.
- **SQL (Structured Query Language)**: Linguagem utilizada para execução dos comandos DDL e DML.
- **Git**: Controle de versão utilizado para gerenciar scripts SQL e outras partes do projeto.

## Funcionalidades Implementadas

- **Criação de Tabela**: Utilização de comandos DDL para criar a estrutura da tabela no banco de dados.
- **Inserção de Dados**: Utilização de comandos DML para inserir registros na tabela.
- **Atualização de Dados**: Utilização de comandos DML para atualizar registros existentes na tabela.
- **Consultas SQL**: Utilização de comandos DML para realizar consultas e verificar os dados na tabela.
